Keyboard Shortcuts

Ctrl+Enter - Add message to history
Ctrl+Shift+Enter - Speak and clear display
Ctrl+K - Clear all messages
Ctrl+B - Toggle babble mode (speak & delete)
Ctrl+P - Toggle speak partial words
Ctrl+, - Open settings
Ctrl+L - Toggle auto-clear after speaking
Ctrl+I - Toggle IPA word lookup
Escape - Close modals

AAC Modes (Require Azure TTS)

👶 Babble Mode (Ctrl+B): Speak each phoneme immediately, then delete it. Perfect for exploring sounds without building text!

🔄 Speak Partial Words (Ctrl+P): Speak the accumulated word after each phoneme. Great for word building practice!

🎤 Speak Button: Speak the current input content immediately.

⚠️ Note: AAC features require Azure TTS API key in settings. Browser speech synthesis cannot properly pronounce IPA phonemes.

IPA Support

Type IPA text directly (no slashes needed): hələʊ or h æ t. See live phoneme images as you type!

Phoneme Images

Live Preview: See phoneme images appear as you type! No need to press Enter.

Display Modes: Click the 🖼️ button to cycle through:

  • 🖼️ Text: Show IPA text only
  • 📝 Images: Show phoneme images only
  • 🔄 Both: Show both text and images

Image File Formats:

CSV Format:

IPA,ImagePath
/h/,phonemes/h.png
/æ/,phonemes/ae.png
/l/,phonemes/l.png
/əʊ/,phonemes/ou.png

JSON Format:

[
  {"ipa": "/h/", "image": "phonemes/h.png"},
  {"ipa": "/æ/", "image": "phonemes/ae.png"},
  {"ipa": "/l/", "image": "phonemes/l.png"},
  {"ipa": "/əʊ/", "image": "phonemes/ou.png"}
]

Note: You can use IPA with or without slashes in your mappings. Both work!

Image Paths: Use relative paths from the HTML file location. Images will be loaded relative to the web app location.

Configuration

Azure TTS (Required for AAC modes): Click Settings to configure Azure TTS API credentials. AAC features (babble mode, speak partial words) require Azure TTS because browser speech synthesis cannot properly pronounce IPA phonemes.

Basic Use: Without Azure TTS, you can still use phoneme images and text, but speaking features will be limited.
